# Dawahi Travel & Tourism Agency - وكالة الدواحي للسفر والسياحة

## نظرة عامة
تطبيق ويب احترافي للسفر والسياحة مع دعم اللغات المتعددة والوضع الليلي.

## الميزات
✨ **واجهة مستخدم:**
- دعم اللغة العربية والإنجليزية
- وضع ليلي وضع نهاري
- تصميم استجابي وحديث
- إطار مضيء للنماذج

🔐 **الأمان:**
- تسجيل دخول وتسجيل آمن
- تشفير كلمات المرور
- التحقق من صلاحية جواز السفر
- حماية المدخلات من الهجمات

✈️ **الحجز:**
- البحث عن الرحلات
- اختيار درجات مختلفة
- حجز الفنادق والغرف
- توليد تذاكر PDF

🏨 **الفنادق:**
- عرض الفنادق والغرف
- تصفية حسب النجوم
- حساب السعر التلقائي

## متطلبات التثبيت
```bash
pip install -r requirements.txt
```

## التشغيل
```bash
python app.py
```

التطبيق سيعمل على `http://localhost:8080`

## هيكل المشروع
```
.
├── app.py                  # التطبيق الرئيسي
├── config.py               # الإعدادات
├── database.py             # إدارة قاعدة البيانات
├── auth.py                 # المصادقة والتحقق
├── constants.py            # البيانات الثابتة والترجمات
├── translations.py         # وظائف الترجمة
├── data.py                 # بيانات البذر
├── requirements.txt        # المكتبات المطلوبة
├── templates/
│   ├── __init__.py
│   └── layout.py           # القوالب الرئيسية
├── utils/
│   ├── __init__.py
│   └── pdf.py              # توليد PDF
├── uploads/                # ملفات جوازات السفر
└── tickets/                # تذاكر PDF
```

## البيانات الافتراضية
يتم إنشاء قاعدة بيانات بـ:
- 25 دولة عربية ودولية
- شركات طيران متعددة
- فنادق بتصنيفات مختلفة (3-5 نجوم)
- رحلات محلية ودولية

## المستخدمون
يمكنك إنشاء حساب جديد عبر صفحة التسجيل مع:
- اسم مستخدم وبريد إلكتروني
- رقم هاتف وجواز سفر
- كلمة مرور آمنة

- الإنجليزية 🇬🇧
